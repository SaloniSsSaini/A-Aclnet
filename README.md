# A-ACLNet

This repository contains the implementation structure for **A-ACLNet**.

## Folder Structure

- data/
  - train/: Training dataset
  - val/: Validation dataset
  - test/: Test dataset

- outputs/
  - models/: Saved models
  - plots/: Training/validation plots
  - heatmaps/: Generated heatmaps

## Usage

```bash
pip install -r requirements.txt
python main.py
```
